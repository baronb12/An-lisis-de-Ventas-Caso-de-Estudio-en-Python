# Anlisis de Ventas Caso de Estudio en Python
Este proyecto es un análisis de datos de ventas completo desarrollado en Python. El objetivo principal fue procesar, limpiar y analizar datos de ventas de una tienda para obtener información estratégica y valiosa.
Descripción del Proyecto
A lo largo del proyecto, se utilizó la librería Pandas para manipular y transformar los datos de ventas. El proceso incluyó los siguientes pasos clave:

Lectura y Fusión de Datos: Se leyeron dos conjuntos de datos de ventas (Datos_Ventas_Tienda.csv y Datos_Ventas_Tienda2.csv) y se concatenaron en un único DataFrame para un análisis unificado.

Tratamiento de Datos: Se aplicaron técnicas de limpieza de datos, como la conversión de tipos de datos (cambiando el formato de la columna de fecha) y la estandarización de valores (corrigiendo la categoría de producto 'Electrónic' a 'Electrónica').

Análisis de Datos: Se realizaron análisis específicos para identificar métricas clave, como el producto más vendido y el mes con mayor cantidad de ventas. Además, se agruparon los datos por categoría de producto para un análisis más profundo de las ventas.

Visualización: Se utilizaron gráficos, incluyendo un gráfico de líneas, para representar visualmente las tendencias de ventas mensuales, haciendo los datos más fáciles de interpretar.

Resultados: El DataFrame final, que incluía una nueva columna con el mes de cada venta, se exportó a un archivo CSV para su almacenamiento y uso posterior.

Este proyecto demuestra habilidades en análisis exploratorio de datos (EDA), limpieza de datos, manipulación de DataFrames con Pandas y visualización de datos con Matplotlib.
